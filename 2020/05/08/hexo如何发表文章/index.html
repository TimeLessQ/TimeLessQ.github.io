hexo发表文章教程

  • 选择一个markdown编辑器

    例如：Typora(个人觉得很好用)

    Typora官网

  • 学会使用makedown写文章（语法很简单，5分钟上手）

    makedown基本语法

  • 将写好的makedown文章复制到你hexo博客文件夹 中 source 下的 _posts 中，默认只有一个 hello-world.md；

  • 复制完之后然后使用文本编辑器打开它，在第一行加上title，内容自定(这个title会成为你文章的标题：不写也没关系，会取默认的作为标题)，

  • 在你博客站点文件夹下执行cmd,打开命令行，输入 hexo g (完整命令为hexo generate)，用于生成静态文件

  • 输入 hexo s(完整命令为hexo server)，用于启动服务器，主要用来本地预览；完成后 打开浏览器输入 localhost:4000，会发现多了你刚写的那篇博客

  • 最后输入hexo d(hexo deploy)，用于将本地文件发布到github等git仓库上
